# ⚡ 빠른 참조 가이드

**30초 만에 핵심 기능 파악하기**

---

## 🎯 이 앱이 하는 일

```
YouTube 채널 데이터 → AI 분석 → 성장 전략
```

---

## 🚀 시작하기 (3단계)

### 1. API 키 발급
```
console.cloud.google.com
→ YouTube Data API v3 활성화
→ API 키 복사
```

### 2. 앱에 입력
```
"API 설정" 탭
→ 키 붙여넣기
→ "테스트" → "저장"
```

### 3. 분석 시작
```
채널 이름 입력 → "분석 시작"
```

---

## 📊 핵심 기능 4가지

### 1️⃣ 채널 분석
```
• 구독자/조회수/참여율
• 최근 영상 퍼포먼스
• AI 전략 보고서
```

### 2️⃣ 키워드 분석
```
• 키워드로 채널 검색
• 경쟁사 비교
• 시장 인사이트
```

### 3️⃣ 라이징 스타 (Pro)
```
• 급성장 채널 발굴
• 성공 패턴 분석
• 벤치마킹 대상 찾기
```

### 4️⃣ 블루오션 (Pro)
```
• 틈새 시장 발견
• 경쟁 낮은 토픽
• 진입 전략 제안
```

---

## 💡 3가지 핵심 팁

### Tip 1: 파레토 차트 활용
```
상위 20% 영상 → 성공 패턴 → 복제
```

### Tip 2: 라이징 스타 벤치마킹
```
급성장 채널 → 공통점 분석 → 적용
```

### Tip 3: 블루오션 진입
```
경쟁 낮은 토픽 → 10개 영상 → 3개월 일관
```

---

## 🔧 자주 쓰는 단축키

### 내보내기
```
JSON: 개발자용
CSV: Excel 분석용
복사: 보고서 작성용
```

### 할당량
```
무료: 하루 10,000 유닛
채널 분석: 약 300 유닛
키워드 분석: 약 200 유닛
```

## ❓ 문제 해결 3초

### API 키 오류
```
설정 → "테스트" → 유효성 확인
```

### 채널 안 나옴
```
정확한 채널명 → 공개 채널만 가능
```

### 할당량 초과
```
다음 날 대기 또는 구매 ($40/100만 유닛)
```
