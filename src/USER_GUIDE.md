**최종 업데이트**: 2025년 11월 5일

## 🎯 서비스 소개

**YouTube 채널 컨설턴트**는 YouTube Data API v3와 AI 인사이트를 결합하여 채널 성장 전략을 제공하는 무료 웹 애플리케이션입니다.

### ✨ 핵심 가치

```
데이터 → 전략 → 실행
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
복잡한 YouTube 분석을
3분 만에 완료하세요
```

## 🌟 주요 기능

### 🆓 Basic 플랜 (무료)

#### 1. **채널 분석** 📊
- 채널의 핵심 지표 (구독자, 조회수, 영상 수)
- 최근 영상 퍼포먼스 분석
- 업로드 패턴 히트맵
- 파레토 차트 (20% 영상이 80% 조회수)
- AI 기반 전략 보고서

#### 2. **키워드 분석** 🔍
- 특정 키워드로 상위 채널 검색
- 경쟁 채널 비교 분석
- 인기 태그 및 키워드 추출
- 시장 기회 발견
- AI 기반 콘텐츠 전략

#### 3. **영상 AI 요약** 🎬
- 개별 영상 상세 분석
- 조회수, 좋아요, 댓글 통계
- AI 기반 영상 요약
- 성공 요인 분석

#### 4. **데이터 내보내기** 📥
- JSON 형식 내보내기
- CSV 형식 내보내기
- AI 보고서 복사
- Excel로 바로 열기 가능

### 💎 Pro 플랜 (무료 체험)

#### 5. **라이징 스타 채널 찾기** ⭐
- 급성장 중인 채널 발굴
- 구독자 대비 높은 조회수
- 최근 3개월 성장률 분석
- 라이징 스코어 자동 계산

#### 6. **블루오션 토픽 분석** 🌊
- 수요는 많지만 공급이 적은 토픽
- 경쟁 강도 vs 검색량 분석
- 틈새 시장 발견
- 진입 전략 제안

## 🚀 빠른 시작 가이드 (5분)

### Step 1: API 키 발급 (3분)

YouTube Data API v3 키가 필요합니다.

#### 1-1. Google Cloud Console 접속
```
https://console.cloud.google.com
```

#### 1-2. 프로젝트 생성
1. **"프로젝트 선택"** 드롭다운 클릭
2. **"새 프로젝트"** 선택
3. 프로젝트 이름 입력 (예: `YouTube Analytics`)
4. **"만들기"** 클릭

#### 1-3. YouTube Data API v3 활성화
1. 좌측 메뉴에서 **"API 및 서비스"** → **"라이브러리"** 클릭
2. 검색창에 `YouTube Data API v3` 입력
3. **"YouTube Data API v3"** 선택
4. **"사용 설정"** 클릭

#### 1-4. API 키 생성
1. 좌측 메뉴에서 **"사용자 인증 정보"** 클릭
2. **"+ 사용자 인증 정보 만들기"** → **"API 키"** 선택
3. API 키 복사 (예: `AIzaSyC-8xwK...`)
4. **(권장)** **"키 제한"** 클릭:
   - **"API 제한사항"** → **"키 제한"**
   - **"YouTube Data API v3"** 선택
   - **"저장"** 클릭

#### 1-5. 할당량 확인
```
무료 할당량: 일일 10,000 유닛
일반적 사용: 하루 20-30회 분석 가능
```

---

### Step 2: 앱에서 API 키 설정 (30초)

#### 2-1. 설정 페이지 접속
```
홈 화면 → "API 키 설정" 탭 클릭
또는
상단 내비게이션 → "API 설정" 클릭
```

#### 2-2. API 키 입력
1. **"YouTube Data API v3 키"** 입력란에 키 붙여넣기
2. **"테스트"** 버튼 클릭
3. ✅ "API 키가 유효합니다!" 메시지 확인
4. **"저장"** 버튼 클릭

**보안**:
- 🔒 브라우저에 **AES-GCM 256-bit 암호화**로 저장
- 🔒 외부 서버 전송 없음
- 🔒 다른 사용자와 공유되지 않음

#### 2-3. 설정 완료 확인
```
상단 헤더에 "● API 연결됨" 녹색 배지 표시
```

### Step 3: 첫 채널 분석 (1분)

#### 3-1. 홈 화면 검색창 이용
1. **홈 화면**으로 이동
2. 검색창에 채널 이름 입력 (예: `Google`, `MrBeast`)
3. **"분석 시작"** 버튼 클릭

#### 3-2. 분석 결과 확인
```
자동으로 채널 분석 페이지로 이동
↓
로딩 (3-5초)

완료! 📊
```

### 3️⃣ 라이징 스타 채널 찾기 (Pro)

#### 접근 방법
```
상단 메뉴 → "공략 채널 찾기"
→ "라이징 스타" 탭
```

#### 라이징 스코어 계산식
```
라이징 스코어 = (최근 조회수 / 구독자 수) × 100

예시:
채널A: 구독자 10K, 최근 평균 조회 50K
→ 스코어 = (50K / 10K) × 100 = 500점
```

#### 발굴 기준
```
✅ 구독자 1K ~ 100K
✅ 최근 3개월 활동 중
✅ 조회수 > 구독자 × 2
✅ 라이징 스코어 > 200
```

#### 화면 구성
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 4️⃣ 블루오션 토픽 분석 (Pro)

#### 접근 방법
```
상단 메뉴 → "공략 채널 찾기"
→ "블루오션 토픽" 탭
```

#### 블루오션 계산식
```
블루오션 지수 = (검색 수요 / 경쟁 강도) × 100

검색 수요 = 키워드 검색 결과 수
경쟁 강도 = 상위 채널 평균 구독자 수
```

#### 발굴 기준
```
✅ 블루오션 지수 > 50
✅ 검색 결과 10개 이상
✅ 상위 채널 구독자 < 100K
✅ 최근 3개월 업로드 있음
```

## 💡 고급 활용 팁

### 📊 Tip 1: 파레토 분석 활용

**목적**: 어떤 영상이 채널 성장에 기여하는지 파악

```
파레토 차트 확인
↓
상위 20% 영상 식별
↓
공통점 분석:
• 썸네일 스타일
• 제목 패턴
• 영상 길이
• 주제 카테고리
↓
성공 패턴 복제
```

**실전 예시**:
```
채널 분석 결과:
상위 20% 영상 = 리뷰 콘텐츠
→ 리뷰 영상 비중 늘리기
```

---

### 🔥 Tip 2: 업로드 히트맵 전략

**목적**: 최적의 업로드 시간 발견

```
업로드 히트맵 확인
↓
경쟁 채널들의 업로드 패턴 분석
↓
비어있는 시간대 공략:
• 경쟁 낮은 시간대
• 타겟 시청자 활동 시간
↓
전략적 스케줄링
```

**실전 예시**:
```
경쟁사: 월/수/금 오전 업로드
나: 화/목 저녁 업로드 → 차별화
```

---

### 🎯 Tip 3: 키워드 조합 전략

**목적**: 경쟁 낮은 롱테일 키워드 발굴

```
메인 키워드 검색 (예: "코딩")
↓
상위 채널 태그 분석
↓
조합 키워드 발견:
• "코딩 + 입문"
• "코딩 + 프로젝트"
• "코딩 + 취업"
↓
각 조합으로 재검색
↓
경쟁 낮은 틈새 발견
```

**실전 예시**:
```
"코딩" → 경쟁 激
"코딩 + 40대" → 경쟁 낮음, 수요 있음
```

### 🚀 Tip 4: 라이징 스타 벤치마킹

**목적**: 빠르게 성장하는 채널의 비밀

```
라이징 스타 검색
↓
스코어 높은 채널 3개 선정
↓
공통점 분석:
